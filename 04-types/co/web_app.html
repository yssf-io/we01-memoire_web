Les sites fonctionnels

Définition : Qu'est ce qu'une web-app ?

Définition Wikipédia

En informatique, une application web (aussi appelée web application, de l'anglais) est une application manipulable directement en ligne grâce à un navigateur web et qui ne nécessite donc pas d'installation sur les machines clientes, contrairement aux applications mobiles. De la même manière que les sites web, une application web est généralement installée sur un serveur et se manipule en actionnant des widgets à l'aide d'un navigateur web, via un réseau informatique (Internet, intranet, réseau local, etc.)

On pourrait le formuler ainsi : "Une web-app est une application destinée à être utilisée sur le web sans rien avoir à télécharger sur la machine personnelle."

Exemple : L'ENT de l'UTC

L'UTC propose tout un système applicatif pour ses enseignents/étudiants afin par exemple de s'inscrire à ses enseignements ou à candidater pour un stage, lire ses mails ou télécharger des supports pédagogiques pour leurs différentes UVs.

Attention : Différence web-app/site web

Définition Wikipédia d'un site web

Un site web, site Web, ou simplement site, est un ensemble de pages web et de ressources reliées par des hyperliens, défini et accessible par une adresse web. Un site est développé à l'aide de langages de programmation web, puis hébergé sur un serveur web accessible via le réseau mondial Internet, un intranet local, ou n'importe quel autre réseau.

Cette courte définition semble englober les applications web car elles aussi sont un ensemble de pages web (en tout cas pour l'utilisateur) accessible par une adresse web.

Cependant si on se réfère à la définition Wikipédia d'une application web que nous avons vu :

En informatique, une application web (aussi appelée web application, de l'anglais) est une application manipulable directement en ligne grâce à un navigateur web et qui ne nécessite donc pas d'installation sur les machines clientes, contrairement aux applications mobiles1. De la même manière que les sites web, une application web est généralement installée sur un serveur et se manipule en actionnant des widgets à l'aide d'un navigateur web, via un réseau informatique (Internet, intranet, réseau local, etc.)

Wikipédia a donc choisi de séparer les deux termes même s'ils auraient alors des similitudes.

Chacun peut avoir son interprétation, comme celle que la catégorie des web app est contenue dans la catégorie des sites web et donc qu'une web-app est un site web et pas l'inverse.

Point low-tech

Ce sont des sites qui visiblement n'ont pas besoin d'une refonte low-tech car ils sont principalement constitués de fichiers JavaScript contenant de la logique métier et des requêtes peu gourmandes à des bases de données. Contrairement aux sites vitrines, leur but premier n'est pas de plaire mais bien d'être utile fonctionnellement parlant.
